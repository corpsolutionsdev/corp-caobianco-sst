## Caobianco SST - Consultoria em Segurança do Trabalho

Site institucional da Caobianco SST com foco em apresentação da empresa,
servicos e contato.

## Visao geral

- Landing page com seções de hero, quem somos, servicos e contato
- Layout responsivo com ajustes para telas pequenas
- SEO basico configurado (canonical, Open Graph, sitemap e robots)

## Stack

- HTML
- CSS
- JavaScript

## Estrutura

- `index.html`: estrutura principal da pagina
- `src/css/base.css`: estilos globais e responsivos
- `src/js/main.js`: interacoes de menu e modal
- `src/img`: imagens e assets

## Como executar

Abra o arquivo `index.html` no navegador ou sirva com um servidor estatico.

## SEO e assets

Arquivos de SEO:

- `manifest.json`
- `sitemap.xml`
- `robots.txt`

URLs base:

- `index.html` (canonical, Open Graph e Twitter)
- `sitemap.xml` e `robots.txt`

Se o dominio mudar, atualize essas URLs.
